    # Contributing to Derek's Reading & Spelling Coach

Thank you for wanting to help improve this project.

This project is being built to help learners who need clear, simple, step-by-step spelling and vocabulary practice.

The goal is to keep the app:

- Simple
- Offline
- Beginner-friendly
- Dyslexia-friendly
- Easy to read
- Easy to understand
- Helpful for learning

## Ways to Help

You can help by:

- Testing the app
- Reporting bugs
- Suggesting new features
- Improving accessibility
- Improving dyslexia-friendly wording
- Adding word lists
- Adding cybersecurity vocabulary
- Improving beginner-friendly comments in the code
- Helping make the app easier for non-technical users

## Keep It Simple

Please keep changes simple and easy to understand.

This project is also a learning project, so clear code and clear explanations are important.

## Accessibility First

When suggesting changes, think about learners who may need:

- Larger text
- Clear instructions
- Less clutter
- Simple menus
- Repetition
- Short steps
- Plain language

## Reporting Problems

When reporting a problem, please include:

- What you were trying to do
- What happened
- What you expected to happen
- Any error message you saw

## Project Mission

The mission is to build a simple spelling and vocabulary coach that helps learners practice at their own pace.
